# Roll A Dice
A fun and simple digital dice-rolling game built for a hackathon!
---
## About
**Roll a Dice** is a lightweight game that stimulates rolling a standard 6-sided die.
Made for quick gamelay and random fun.
---
## Features
- Generates a random number from **1-6**
- Unlimited rolls
-  Simple and clean interface
-  Fast and responsive
- Alerts for win or lose
---
## How to Play
1. Open the game on a web browser usign this link - [Click here!](https://samcooper7.github.io/Hackathon-Game-Roll-a-dice/ "The best game in the world")

2. Click the **Roll** button.
3. The die will show a random number between 1 and 6.
4. If you score above 12 you win, if not you roll again!
